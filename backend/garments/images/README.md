# Garment Images

Place your garment images here with the following naming convention:

## Required Images:

- `shirt_001.jpg` - White Casual Shirt
- `tshirt_001.jpg` - Black T-Shirt
- `kurta_001.jpg` - Blue Kurta

## Image Requirements:

- Format: JPG or PNG
- Recommended size: 512x512 or larger
- Background: Plain white or transparent
- Subject: Garment should be centered and clearly visible
- Quality: High resolution for best try-on results

## How to Add Images:

1. Download or prepare your garment images
2. Rename them according to the IDs in `catalog.json`
3. Place them in this folder
4. Update `catalog.json` if adding new garments

## Testing:

You can use placeholder images from Unsplash or similar sources for testing.
The `thumbnail_url` in catalog.json is used for display in the app.
The `image_path` points to the actual garment image used for try-on.
